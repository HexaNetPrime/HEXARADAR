## 📸 What is HEXARADAR?

HEXARADAR is a **professional network security scanner** that helps you:
- 🔍 **Network Scanning** - Nmap-based scanning with 60+ options
- 🤖 **AI Vulnerability Detection** - Predict vulnerabilities using Machine Learning
- 🌐 **Network Mapping** - Discover hosts, topology, dependencies
- 🔍 **OSINT** - Domain intelligence, WHOIS, DNS, Email enumeration
- 📊 **Data Analytics** - Trend analysis, Risk scoring, Compliance checks
- 🛡 **Evasion Techniques** - Proxy chains, VPN, MAC randomization

> **Perfect for:** Security Researchers, Network Admins, Penetration Testers
