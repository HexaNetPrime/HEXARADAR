1. 📖 Introduction

EVASION TECHNIQUES Tab आपको Firewall, IDS, IPS को बायपास करने में मदद करता है। जब आप किसी network पर scan करते हैं, तो Firewall/IDS आपको detect कर सकता है और आपका IP block कर सकता है। ये techniques आपको detection से बचने में मदद करती हैं।

💡 कब USE करें?
• जब आपको संदेह हो कि Firewall/IDS active है
• जब आप अपनी real identity hidden रखना चाहते हैं
• जब आपको किसी protected network को scan करना हो
• जब आप penetration testing कर रहे हों
⚠️ LEGAL DISCLAIMER:
इन techniques का use केवल authorized systems पर करें। बिना permission के किसी के network पर scan करना illegal हो सकता है।

3. 🌐 PROXY CHAINS

🎯 क्या करता है:
Tor/SOCKS5/HTTP proxy के through scan करता है ताकि आपकी real IP hidden रहे

📋 Step-by-Step Guide:

STEP 1: Tor Proxy Install करें
# Linux
sudo apt install tor
sudo systemctl start tor

# Windows: https://www.torproject.org/download/
STEP 2: EVASION Tab में Proxy Configure करें
Type: SOCKS5 (Select करें)
Host: 127.0.0.1 (Default Tor host)
Port: 9050 (Default Tor port)
STEP 3: ▶ ENABLE दबाएं
✅ Proxy Enabled: socks5://127.0.0.1:9050
💡 Nmap Flag: --proxies socks5://127.0.0.1:9050
STEP 4: Scan करें
MAIN Tab → Target डालें → ▶ RUN दबाएं

# Manual Command भी use कर सकते हैं:
nmap --proxies socks5://127.0.0.1:9050 192.168.159.128
STEP 5: ⏹ DISABLE दबाएं
Scan के बाद proxy disable करें

4. 🔒 VPN ROTATION

🎯 क्या करता है:
VPN status check करता है और VPN को restart (rotate) करता है ताकि IP change हो जाए

📋 Step-by-Step Guide:

STEP 1: VPN Interface Configure करें
Interface: tun0 (OpenVPN के लिए) या wg0 (WireGuard के लिए)
Provider: OpenVPN / WireGuard
STEP 2: 🔍 CHECK STATUS दबाएं
🔒 VPN STATUS CHECK
✅ VPN Interface: tun0 - UP
📍 VPN IP: 10.8.0.2
STEP 3: 🔄 ROTATE VPN दबाएं
🔄 VPN ROTATION
🔄 Rotating OpenVPN VPN on tun0
✅ New IP: 10.8.0.5
✅ VPN Rotated Successfully!
STEP 4: अब scan करें
नई IP से scan करें (IP change हो गई है)
⚠️ NOTE:
VPN rotate करने के लिए आपके system में VPN configured होना चाहिए।

5. 🔧 MAC RANDOMIZATION

⚠️ WARNING:
यह feature sudo/root permission चाहिए! Network interface down हो सकता है।
🎯 क्या करता है:
Network interface का MAC address बदलता है ताकि network पर आपकी identity hidden रहे

📋 Step-by-Step Guide:

STEP 1: Interface Name पता करें
# Terminal में
ip link show
# या
ifconfig

# Output: eth0, wlan0, ens33, आदि
STEP 2: EVASION Tab में Interface डालें
Interface: eth0 (या आपका interface name)
STEP 3: 🔀 RANDOMIZE MAC दबाएं
🔧 MAC RANDOMIZATION
📌 Original MAC: 00:11:22:33:44:55
🔀 New MAC: 02:00:00:12:34:56
✅ MAC Randomization Successful!
STEP 4: Scan करें
अब scan करें (नए MAC address के साथ)
STEP 5: ↩️ RESTORE MAC दबाएं
↩️ RESTORE MAC
✅ MAC Restored: 00:11:22:33:44:55
⚠️ IMPORTANT:
• MAC change करने के बाद network connection temporarily disconnect हो सकता है
• Always restore MAC after scan
• Virtual machines (VMware/VirtualBox) में भी काम करता है

6. 📊 TRAFFIC SHAPING

🎯 क्या करता है:
Scan की speed को control करता है ताकि IDS/IPS detect न करें

📋 Step-by-Step Guide:

STEP 1: Parameters Set करें
Scan Delay (ms): 500 (हर packet के बीच 500ms delay)
Max Rate (pkts/sec): 1000 (ज्यादा से ज्यादा 1000 packets/sec)
STEP 2: ✅ APPLY SHAPING दबाएं
📊 TRAFFIC SHAPING
✅ Traffic Shaping Applied
📊 Scan Delay: 500ms
📊 Max Rate: 1000 packets/sec
💡 Nmap Flags: --scan-delay 500ms --max-rate 1000
STEP 3: Scan करें
MAIN Tab → Target डालें → ▶ RUN दबाएं

# Manual Command:
nmap --scan-delay 500ms --max-rate 1000 192.168.159.128

8. 🎯 Complete Workflow

Detection से बचने के लिए पूरा process:

STEP 1: 🕵️ IDS/IPS DETECTION चलाएं
पता करें Firewall/IDS है या नहीं
STEP 2: अगर IDS/IPS Active है:
├── 🌐 PROXY ENABLE करें (Tor के through)
├── 📊 TRAFFIC SHAPING Apply करें (slow down)
├── 🔧 MAC Randomize करें (identity hide)
└── 🔍 अब Scan करें (detection से बचने के लिए)
STEP 3: Scan करें
MAIN Tab → Target डालें → ▶ RUN दबाएं
STEP 4: Scan के बाद Cleanup करें:
├── 🌐 PROXY DISABLE करें
├── 🔧 MAC Restore करें
└── 📊 Traffic Shaping Off करें
✅ EXAMPLE: Complete Command
# Step 1: Detect IDS
→ 🕵️ IDS/IPS DETECTION दबाएं

# Step 2: Enable Proxy
→ 🌐 PROXY: Type SOCKS5, Host 127.0.0.1, Port 9050 → ▶ ENABLE

# Step 3: Apply Traffic Shaping
→ 📊 TRAFFIC: Delay 500ms, Max Rate 500 → ✅ APPLY SHAPING

# Step 4: Scan
→ MAIN Tab → Target → ▶ RUN

# Step 5: Cleanup
→ EVASION Tab → 🌐 PROXY DISABLE → 📊 Traffic Shaping Off

9. ❓ FAQ

Q1: Proxy Enable करने के बाद scan क्यों नहीं हो रहा?
Tor proxy सही से running है या नहीं check करें। sudo systemctl status tor
Q2: MAC Randomize करने पर network disconnect क्यों हो गया?
MAC change करने पर network interface temporarily down हो जाता है। कुछ सेकंड wait करें, auto reconnect हो जाएगा।
Q3: VPN Rotate क्यों काम नहीं कर रहा?
VPN properly configured है या नहीं check करें। OpenVPN/WireGuard installed होना चाहिए।
Q4: IDS/IPS Detection में कोई result क्यों नहीं आ रहा?
Target reachable है या नहीं check करें। Firewall ICMP को block कर रहा हो सकता है।
Q5: Traffic Shaping से scan slow क्यों हो रहा है?
यह intentionally slow है। IDS/IPS detection से बचने के लिए scan deliberately slow किया जाता है।
