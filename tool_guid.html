11. ❓ FAQ

Q1: AI Features क्यों काम नहीं कर रहे?
पहले scan करें (MAIN Tab से), फिर AI features use करें। AI features scan results के बिना काम नहीं करते।
Q2: Proxy Enable करने के बाद scan क्यों नहीं हो रहा?
Tor proxy सही से running है या नहीं check करें: sudo systemctl status tor
Q3: MAC Randomize करने पर network क्यों disconnect हो गया?
MAC change करने पर network interface temporarily down हो जाता है। कुछ सेकंड wait करें, auto reconnect हो जाएगा।
Q4: Compliance FAIL क्यों आ रहा है?
यह ERROR नहीं है, यह actual result है। Telnet, FTP, SMB जैसे insecure services चल रहे हैं, इसलिए FAIL आ रहा है।
Q5: Data Analytics के लिए कितने scans चाहिए?
• TREND के लिए: कम से कम 2 scans
• PREDICT के लिए: कम से कम 5 scans (अलग-अलग दिनों में)
